A quantum interpreter in BQN

Preamble

We start by importing the required system functions and defining a modifier to work with complex matrix products. Then we create a namespace with some quantum gates: 

Sin‿Cos‿GCD ← •math
U ← •rand.Range
_cp ← {(-´𝔽¨)⋈(+´𝔽¨)⟜⌽}

g ← {
  IM ← (⊢⋈≢⥊⟜0⊢)¨
  id‿x‿h ⇐ (⌽‿⊢{𝕎𝕩}¨<=⌜˜↕2) ∾○IM <-⌾(1‿1⊸⊑)2‿2⥊÷√2
  swap‿cnot ⇐ IM ⟨1‿2, 2‿3⟩ {⌽⌾(𝕨⊸⊏)𝕩}¨ <=⌜˜↕4
  P ⇐ 3‿3⊸⊑{(Sin⌾𝔽⋈Cos⌾𝔽) 𝕩⌾𝔽 =⌜˜↕4}
}

M‿K‿D ← (+˝∘×⎉1‿∞ _cp)‿(∾⊣×·<⊢ _cp)‿{1+⌊2⋆⁼⊑𝕩}

Interpreter

The (415 chars) quantum interpreter is based on references arXiv:1711.02086 and arXiv:1608.03355. For simplicity, we always measure at the end of the execution: 

Q ← {𝕊qb‿sc:
  psi ← {1⌾⊑𝕩⥊0} 2⋆qb
  M‿K‿D ← (+˝∘×⎉1‿∞ _cp)‿(∾⊣×·<⊢ _cp)‿{1+⌊2⋆⁼⊑𝕩}
  L ← {K´ ⟨(qb-𝕨-D≠𝕩) {K⍟(𝕨-1)˜ 𝕩} g.id, 𝕨 {K⍟(𝕨-1)˜ 𝕩} g.id, 𝕩⟩}
  T ← ∾↕∘≠{a←𝕩 ⋄ i←𝕨{𝕩⊑a}•_while_{𝕩<𝕨}𝕨⊑a ⋄ 𝕨<◶⟨⟩‿{(⊢∾1⊸↓∘⌽)𝕨+↕𝕩-𝕨}i}¨<
  A ← {v𝕊qs‿gs:
    1=◶{
      ui ← 0 L gs
      v M˜ {0=≠𝕩 ? ui; {ui 𝕨⊸M∘M 𝕩}´ (M˜∘⌽⋈M)´ {𝕩 L g.swap}¨ 𝕩} T qs (⌽∘⊢∾¬∘∊/⊣)˜ ↕qb
    }‿(v M˜ qs L gs) ≠⊏qs}
  »⊸<∨` 0(-`˜⟜U˜<⊣) +○(×˜)˝ psi A˜´ ⌽sc
}

Shor's algorithm

As a test case, we employ the quantum circuit of Shor's algorithm for the number fifteen and base eleven, following references arXiv:1804.03719 and arXiv:2306.09122. The resulting compiled circuit uses five qubits, three of which serve as control. To enhance statistical accuracy, the experiment is repeated multiple times. Additionally, we define a classical post-processing function: 

n‿a‿qb‿r ← ⟨15, 11, 5, 5⟩

sc ← ⟨
  ⟨0⟩‿g.h ⋄ ⟨1⟩‿g.h ⋄ ⟨2⟩‿g.h
  ⟨2, 3⟩‿g.cnot ⋄ ⟨2, 4⟩‿g.cnot ⋄ ⟨1⟩‿g.h
  ⟨⟨1, 0⟩, g.P π÷2⟩ ⋄ ⟨0⟩‿g.h
  ⟨⟨1, 2⟩, g.P π÷4⟩ ⋄ ⟨⟨0, 2⟩, g.P π÷2⟩ ⋄ ⟨2⟩‿g.h
⟩

Epilogue

Why BQN, you may ask? Its tacit and functional style together with its support for numerous combinators, which makes programming feel like solving an algebraic problem rather than drafting a bureaucratic manifesto. Try running the simulation in the BQN repl and explore it! While the interpreter is not particularly optimized, here is a comparison with Common Lisp for those who are curious:
